# Stm32NetXMqttClient

A MQTT client for Stm32NetX



## Requirements

* ThreadX (Core) (Memory pool size: 5*1024)
* NetX (NX Core, Ethernet Interface, Ethernet Phy Interface) (Memory pool size: 35*1024)
* HAL ethernet configuration (MAC address?)
* Stm32ItmLogger
* Stm32Common
* Stm32ThreadX
* Stm32NetX
